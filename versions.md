## [0.2.0] - 2025-12-29

Summary
- Consolidate Copilot-agent and owner changes that add the Fishing Time page, chart controls, animations, and documentation. This release groups the merged work that landed on and before 2025-12-29.

### Added
- Add Fishing Time page (`fishing.html`) to provide a dedicated fishing visualization and interactive controls.
- Add fishing-related scripts and modules:
  - `fishing-script.js` — page scripting and UI glue.
  - `fishing-animation.js` — underwater and surface animation, tide markers, sun rendering.
  - `marine-species.js` — marine species data and accordion table integration.
  - `tide-harmonic.js` — tide harmonic calculation utilities.
- Add documentation and design notes:
  - `IMPLEMENTATION_SUMMARY.md` — comprehensive implementation summary (Co-authored by Copilot agent).
  - `FISHING_FEATURES.md`, `COMPONENTS.md` — feature and component documentation.
- Add chart and visualization scaffolding:
  - Insert canvas elements for charts on the fishing page.
  - Add dataset controls and sync sliders for interactive data selection.
- Add UI components:
  - Add slide-out menu, 50/50 layout, sticky header navigation, and map-based location selection elements.
  - Add accordion fish species table for species lookup.

### Changed
- Refactor `fishing.html` layout and reorganize page structure to reduce cross-page JavaScript errors.
- Rework site styles for a polished, professional theme and remove duplicate rules in `styles.css`.
- Normalize stacked chart rendering and add consistent chart initialization timing.
- Integrate interactive time slider with tide harmonic calculations into the visualization flow.
- Extract magic numbers into constants and improve inline documentation.
- Clean up comments and remove temporary development artifacts.

### Fixed
- Fix chart canvas ID mismatch and add missing render triggers to ensure charts render reliably.
- Remove extra closing brace in `script.js` that caused runtime errors.
- Rename animation variable from `direction` to `windDir` to correct wind-related animation.
- Fix CSS `@keyframes` syntax (remove invalid spacing) to restore animations.
- Address script initialization issues and improve usage patterns flagged during reviews.
- Improve validation and error handling, and document expected data formats.

### Deprecated
- None in this release.

### Removed
- Remove right-side controls (simplify fishing page layout).
- Remove duplicate CSS and debug-only code used during prototyping.

### Security
- No security issues reported in the merged commits used for this release.
